#  E-commerce Sales Data Analysis

##  Overview
This project analyzes a real-world e-commerce dataset to extract business insights related to revenue, customer behavior, and delivery performance.

##  Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

##  Workflow
- Data Cleaning
- Data Merging
- Feature Engineering
- Exploratory Data Analysis
- Data Visualization

##  Key Insights
- Revenue is driven by a few top categories
- Delivery delays impact customer experience
- Most users prefer specific payment methods

##  Business Recommendations
- Focus marketing on top categories
- Improve delivery efficiency
- Optimize payment strategies

##  Output
Cleaned dataset and analysis notebook included.
